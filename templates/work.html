001 : Python projects and resources

Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

The Python language has emerged as one of the best and most popular language over the years. Prior to python i used to code in C, C++ and Java. The python codes are comparatively easy to read, understand and write. Open source and vast libraries makes python one of the best language.


This page contains python projects made by me while learning this beautiful language.


1. Password checker

The application allows user to enter the password from command line while running the python script and displays how many times that password has been cracked.

The application uses the REST API provided by the haveibeenpwned.com. The API allows the list of pwned accounts (email addresses and usernames) to be quickly searched via a RESTful service.

2. IMDB Movie Ratings

The application allows user to scrap the IMDB website and retrieve top 250 movies as per their IMDB ratings.

The application uses the Python Web Scrapping to retrieve the raw response from IMDB and then python functions are used to extract the data in a desired format.
