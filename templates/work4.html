004 : Network Automation tools and resources

Computer networking has been same for almost 3 decades. However with services like cloud and virtualization, the industry is moving towards automated world. The static world of networking is rapidly moving towards a dynamic world where networks and devices will be controlled by codes.

A new era of networking has begun and mastering the automation of networking devices has become need of an hour.

Automation tools have been adding great value to the industry but the core networking is still using same old Vender specific devices and protocols. This scenario is also changing now as we are moving towards software defined networking (SDN). The SDN network has separate data plane and control plane and all the forwarding decisions are made by controllers. This eliminates the use of specific vendors as the forwarding logic is now implemented in the form of software controllers. The SDN allows more flexible service as we are free to control everything as per the requirements. .

This page covers Automation of networking deviecs like Routers, switches and Servers suing python code and SDN Technology explored by me.
